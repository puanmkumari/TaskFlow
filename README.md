# 🌐 TaskFlow

A responsive personal portfolio website built using **HTML5** and **CSS3**. The website contains multiple pages and is fully responsive for desktop and mobile devices.

## ✨ Features

* Home, About & Contact Pages
* Responsive Navigation Bar
* Flexbox & CSS Grid Layout
* Mobile-Friendly Design
* Hover Effects
* Clean UI

## 🛠️ Technologies

* HTML5
* CSS3
* Flexbox
* CSS Grid
